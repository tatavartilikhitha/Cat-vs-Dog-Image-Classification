# 🐱 Cat vs Dog Image Classification

This project classifies Cat and Dog images using a Convolutional Neural Network (CNN) built with TensorFlow/Keras.

## Features
- Train CNN model
- Predict Cat or Dog images
- Tkinter GUI
- TensorFlow & Keras

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Pillow
- Tkinter

## Dataset
Kaggle Cats vs Dogs Dataset
